# 🌐 Language Translation Tool

## CodeAlpha AI Internship - Task 1

### 📖 Project Overview

The Language Translation Tool is a web application developed using **Python** and **Flask**. It allows users to translate text between multiple languages using the **deep-translator** library through a simple and user-friendly interface.

### 🚀 Features

* Translate text between multiple languages
* User-friendly web interface
* Select source and target languages
* Instant translation results
* Built using Flask and Python

### 🛠️ Technologies Used

* Python
* Flask
* HTML5
* CSS3
* deep-translator

### ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/varun-git-h/CodeAlpha_LanguageTranslationTool.git
```

2. Navigate to the project folder

```bash
cd CodeAlpha_LanguageTranslationTool
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```
http://127.0.0.1:5000
```
